# Education Analytics Dashboard

This project is a Power BI dashboard that analyzes student performance using education analytics.

## Features
- Student performance analysis
- Attendance impact on results
- Study hours vs scores analysis
- Parent education impact
- Interactive slicers and filters

## Tools Used
- Power BI
- Data Visualization
- Kaggle Dataset
- GitHub

## Dashboard Insights
- Higher study hours improve student performance
- Attendance significantly affects academic results
- Internet access and parent education influence outcomes
